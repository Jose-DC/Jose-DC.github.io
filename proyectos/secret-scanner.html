Idea inicial

Secret scanner

Ficha inicial de una herramienta CLI en Python para revisar repositorios antes de publicarlos.

Estado

Todavía no está implementado.

Objetivo

Detectar archivos o patrones sensibles como `.env`, claves privadas, tokens y posibles credenciales antes de hacer commit o push.

Tecnologías

  • Python
  • Git
  • Markdown para reportes

Qué documentaré

  • Reglas de detección.
  • Ejemplos de uso.
  • Casos de prueba sin información real.
  • Limitaciones y falsos positivos.
