Proyecto 03

Revisión de logs

Ficha inicial de un script para apoyar tareas de soporte y troubleshooting sobre archivos de log.

Estado

Planeado. Todavía no está implementado.

Objetivo

Leer logs, detectar errores frecuentes, agrupar mensajes relevantes y generar un resumen útil para diagnóstico o escalamiento.

Tecnologías

  • Python
  • Linux
  • Archivos de log
  • Markdown para reportes

Qué documentaré

  • Patrones detectados.
  • Ejemplos de logs ficticios.
  • Formato de reporte.
  • Casos donde el script ayuda y donde no.
